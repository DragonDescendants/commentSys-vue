# 项目概述

这是一个使用[Vue3](https://vuejs.org/)和[Vite](https://vitejs.dev/)构建的商品评价系统的前端页面

## 组成部分

1. 首页的商品展示

2. 用户的个人中心
    1. 个人信息的设置与修改
    2. 个人问答评价讨论查看
    3. ...

3. 商品的详细信息展示
    1. 商品信息页
    2. 商品问答页
    3. 商品评价讨论页
    4. ...

4. 管理员管理页面
    1. 管理评价讨论
    2. 管理提问回答
    3. 设置商品标签
    4. ...

## 使用的组件
[Element-plus](https://element-plus.org/en-US/)
[pinia](https://pinia.vuejs.org/)
[axios](https://axios-http.com/)
